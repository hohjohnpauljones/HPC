All testing and analysis was done on Babbage2

to run do the following:

1. make clean

2. make

3. ./hw2 <filepath>

To change program parameters, see the variable declaration section at the 
beginning of main in hw2.cpp. 

int D = 10;						//SETS THE NUMBER OF TOP RESULTS TO FIND
int P = 1;						//SETS THE DEGREE OF MULTIPROCESSING
int sizes[] = {9, 11, 17, 29};	//SETS THE SIZES OF THE SEARCH VECTORS
